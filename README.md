## 주요 기능

- **Hero 섹션** — 프로필 소개 및 주요 링크
- **Skills** — 기술 스택 원형 차트 (6개 카테고리)
- **Career** — 경력 타임라인
- **Projects** — 회사/개인 탭 필터링, 모달 상세보기, 페이지별 탭 UI
- **다크/라이트 테마** 토글
- **반응형 디자인**

## 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Theme**: next-themes
